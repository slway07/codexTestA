# Tarot MVP (No-LLM First)

이 저장소는 **LLM 없이 시작하는 타로 웹사이트 MVP**를 구현합니다.

## 현재 구현 범위 (MVP v2)
- 원카드 / 3카드(과거·현재·미래) 스프레드 선택
- 카드 섞기 후 펼쳐진 덱에서 직접 카드 선택
- 메이저 아르카나 22장 기반 랜덤 리딩
- 정/역방향 랜덤 결정
- 분야별(연애/일·커리어/금전/일반) 템플릿 해석
- 최근 리딩 기록 10개 로컬 저장(localStorage)

## 실행 방법
정적 파일만 사용하므로 브라우저에서 `index.html`을 열면 실행됩니다.

간단히 로컬 서버를 띄우려면:

```bash
python3 -m http.server 4173
```

그 다음 브라우저에서 `http://localhost:4173` 접속.
